# Contributing Guide

Dự án này là dự án phát triển cá nhân/nội bộ, không áp dụng các quy trình Git Flow hay Review Code phức tạp.
Toàn bộ quy tắc về cấu trúc phát triển và nguyên tắc code đã được rút gọn lại để AI hoặc nhà phát triển chính tự do tối ưu hóa.

Vui lòng xem thông tin kiến trúc ngắn gọn tại [AGENTS.md](./AGENTS.md).
